# SmsSendingBundle

SmsSendingBundle is a Symfony 3.4 bundle for tunisiesms Api .

## config.yml

Use this line in config.yml file.

```yml
sarbacane:
    key: '%smsKey%'
    sneder: '%sender%"
    mobile: '%mobile%'
```

## parameters.yml

```yml
sms_key: [YOUR_SMSKEY]
sender: [YOUR_SENDER_NAME]
mobile: [YOUR_MOBILE_NUMBER]
```
